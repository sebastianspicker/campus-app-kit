# Code of Conduct

We want this project to be a safe, welcoming place to contribute.

## Our standards

Examples of behavior that helps the project:

- Be respectful and direct.
- Assume good intent and ask clarifying questions.
- Keep discussions technical and focused on outcomes.

Examples of unacceptable behavior:

- Harassment, hate speech, or personal attacks.
- Publishing someone else’s private information.
- Deliberately introducing security issues or secrets.

## Enforcement

If you experience or witness unacceptable behavior:

- If it’s safe, open a GitHub issue describing what happened.
- If it includes sensitive information, use a GitHub Security Advisory instead of an issue.

Project maintainers may remove content or restrict participation to keep the community safe.
