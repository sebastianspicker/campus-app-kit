# Roadmap

This roadmap tracks the public template only. Private connectors and ops live elsewhere.

## Short term

- Add basic UI polish for Today / Rooms / Stage.
- Expand sample data and fixtures for BFF routes.
- Improve error handling and offline states.

## Mid term

- Add typed API client generated from shared schemas.
- Introduce real caching strategy for BFF responses.
- Add basic accessibility checklist to the mobile app.

## Long term

- Optional multi-institution switching in the app.
- Localization support for campus labels and events.
- Extend public pack schema with optional categories and tags.
